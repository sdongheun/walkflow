# Walkflow MVP Specification

## 1. 개요

현재 MVP는 집에서부터 학교까지 길찾기 시, 공공데이터(횡단보도 신호 주기)와 사용자의 실시간 보행 속도(건강 앱 연동)를 결합하여 "횡단보도 대기 시간을 최소화하는 최적의 출발 시간"을 알려주는 앱입니다. 나만의 걷기 비서처럼 횡단보도 대기 시간 없이 목적지에 도착할 수 있도록 타이밍을 맞춰줍니다.

## 2. 주요 기능

### 1) 데이터 수집 (User Profile)

- **건강 앱 연동**: 사용자의 최근 1개월 평균 보행 속도(m/s) 가져오기.
- **기본값 설정**: 권한 거부 시 성인 평균 속도(1.2m/s)로 자동 세팅.

### 2) 경로 및 신호 분석 (Path & Signal)

- **단일 경로 탐색**: 현재 위치에서 목적지까지 최단 거리 횡단보도 1개 추출.
- **신호 데이터 매칭**: 해당 횡단보도의 신호 주기(초록불 시간, 빨간불 시간) 가져오기.
- **시간 계산**: (내 위치 ~ 횡단보도 거리) / (내 보행 속도) = 도착 예정 시각.

### 3) 최적 시간 알림 (Core UX)

- **타이밍 계산**: 도착 예정 시각이 횡단보도의 '초록불 시작 시간'과 일치하는 가장 가까운 출발 시각 도출.
- **결과 화면**: "지금 바로 출발하세요!" 또는 "[N]분 [M]초 뒤에 출발하면 기다리지 않고 건넙니다."라는 메시지 표시.

## 3. 추천 기술 스택 및 라이브러리 (App)

- **Framework**: `React Native (Expo)` - 가장 빠른 빌드 및 건강 앱/지도 생태계 활성화.
- **State Management**: `Zustand` - 가볍고 빠른 사용자 설정/데이터 전역 저장.
- **Database / BaaS**: `Supabase` - 실시간 신호 주기 캐싱 및 사용자 설정 저장.
- **Maps**: `Kakao/Naver Map SDK`, `react-native-maps` - 국내 횡단보도 좌표 및 보행자 경로 데이터 시각화.
- **Traffic Data**: `서울시/공공데이터포털 API` - 실시간 신호 주기 데이터 통신.
- **Health Data**: `react-native-health` (iOS HealthKit) / `react-native-google-fit` (Android) - 건강 앱 데이터 추출.
